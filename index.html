¿Cómo usar las plantillas de armaduras?

Una vez que hayas obtenido las plantillas de herrería, puedes usarlas para crear armaduras y armas únicas. Para usar las plantillas, debe tener los materiales y recursos necesarios para fabricar la armadura y las armas.
Para fabricar la armadura y las armas usando las plantillas, debe colocar la plantilla en el yunque y luego ponerle los materiales necesarios. El yunque luego creará la armadura o el arma, y podrás usarlo en tu juego.
